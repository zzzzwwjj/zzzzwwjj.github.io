看完题目后以为题目中的 非空不重叠 子数组即每个数不能用两次的意思，按照这个理解想了很久都没想到什么很好的题解，只好看评论和题解，发现原来这个所谓的 非空不重叠 子数组是指每个子数组的区间范围不重叠。。。

示例： 输入： nums = [2, 6, 8, 4], target = 10 按我原来的理解应该输出为 2 ，然而实际预期结果为 0 ，因为子数组区间重叠了。

这样一来这道题就很简单了。

class Solution {
    public int maxNonOverlapping(int[] nums, int target) {
        Set<Integer> set = new HashSet<>();
        set.add(0);
        int sum = 0;
        int result = 0;
        for(int num: nums) {
            sum += num;
            if(set.contains(sum-target)) {
                set.clear();
                set.add(0);
                sum = 0;
                result++;
            } else set.add(sum);
        }
        return result;
    }
}

感觉我原来理解的题目是个NP问题。。。
